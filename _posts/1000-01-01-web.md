# Jekyll

## Jekyll installation

```
sudo apt-get -yqq install ruby ruby-dev make nodejs
sudo gem install --no-rdoc --no-ri jekyll
```

## Build and server

```
sudo jekyll build /var/www/html
sudo jekyll serve -H "0.0.0.0" -P 4000
```
