## Installation and configurations

```
sudo apt-get install tftp-hpa tftpd-hpa
```

```
cd /
sudo mkdir /tftpboot
sudo chmod 777 /tftpboot
sudo in.tftpd -l /tftpboot
```

## Run it

```
cd /tftpboot
touch test
```

```
cd /home/usrname
tftp 192.168.0.59
```

```
get /tftpboot/test
```

## Troubleshooting

* Cannot put file on it

```
cd /etc/xinetd.d/
```

New tftp file:

```
service tftp
{
disable = no
socket_type = dgram
protocol = udp
wait = yes
user = root
server = /usr/sbin/in.tftpd
server_args = -s /tftpboot -c
per_source = 11
cps = 100 2
}
```

```
sudo /etc/init.d/xinetd restart
sudo in.tftpd -l /tftpboot
```
